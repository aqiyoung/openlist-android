## 关键 bug 修复
1. **文件列表空** (v0.2.3): `FsItem.isDir` 加 `@SerialName("is_dir")` — OpenList 4.x 字段是 snake_case
2. **登录不跳转** (v0.2.2): onLoginSuccess 空 stub 改 `setLoggedIn()`
3. **下载 401** (v0.3.3): 去 "Bearer " 前缀 — OpenList 4.x 中间件不认 Bearer prefix
4. **上传 storage not found** (v0.3.3): 根目录上传抛 error 提示进 storage
5. **HTTP 413** (nginx 06-14 00:40): 加 `client_max_body_size 0` 在 location / 段
